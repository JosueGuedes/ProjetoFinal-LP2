# ProjetoFinal-LP2

### Entrega 1: Logging, headers e diagrama
Esta primeira entrega do projeto final da disciplina de Linguagem de Programação II envolveu primariamente a criação de uma biblioteca pessoal para logging que seja thread-safe através do uso de mutexes.
O tópico escolhido para desenvolvimento foi o tópico A: O servidor de chat multiusuário em TCP.
